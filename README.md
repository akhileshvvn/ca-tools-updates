# ca-tools-updates

CA Tools 1.0.0 – PDF Table Extractor

Overview
CA Tools is a Windows desktop utility by VVN Tech Labs to simplify day‑to‑day compliance workflows for CAs and finance teams. Version 1.0.0 introduces the first module: PDF Table Extractor.

What’s new in 1.0.0

New: PDF Table Extractor

Select any PDF and extract all detected tables into a single Excel workbook, each table on its own sheet.

Modern, lightweight desktop UI (no browser or local Python setup required).

Supports saving directly to a user‑chosen Excel file path.

Packaging & updates

Bundled as a standalone CA-TOOLS.exe (no dependency installation on client machines).

Auto‑updating launcher (CA-TOOLS-Launcher.exe) checks for new versions and downloads them automatically.

Known limitations

Designed and tested for Windows 10/11, English locale.

Complex or heavily formatted PDFs may yield imperfect table structures; always review critical outputs.
